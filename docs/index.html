ᨒ ᨁᨒᨗᨁᨚ (NBG-188)

ᨌᨙᨉᨗ

ᨕᨗᨕ ᨊᨕᨙ ᨄᨆᨘᨒᨊ ᨔᨘᨑᨛ ᨁᨒᨗᨁᨚᨕᨙ᨞ ᨑᨗ ᨔᨓᨙᨑᨗᨁᨉᨗ᨞

ᨑᨗᨕᨘᨒᨚᨊ ᨅᨈᨑ ᨁᨘᨑᨘ ᨑᨗ ᨒᨘᨓᨘ᨞
ᨈᨚᨇᨚᨊ ᨓᨙ ᨎᨗᨒᨗ ᨈᨗᨆᨚ ᨑᨗ ᨓᨈ ᨇᨑᨛ᨞
ᨑᨗᨄᨚᨓᨓᨗᨊᨙᨊ ᨓᨙ ᨎᨗᨒᨗ ᨈᨗᨆᨚ ᨑᨗ ᨅᨈᨑ ᨁᨘᨑᨘ᨞
ᨆᨂᨗᨉᨛᨊ ᨓᨙ ᨎᨗᨒᨗ ᨈᨗᨆᨚ᨞
ᨍᨍᨗᨊ ᨅᨈᨑ ᨒᨈᨘ᨞
ᨑᨗᨄᨕᨙᨋᨙᨊ ᨑᨗ ᨈᨚᨍ ᨅᨈᨑ ᨒᨈᨘ᨞
ᨑᨗᨄᨒᨙᨍᨊ ᨑᨗ ᨈᨊ ᨅᨈᨑ ᨒᨈᨘ᨞

ᨆᨊᨘᨑᨘᨊ ᨒ ᨕᨘᨑᨘ ᨇᨛᨔᨗ ᨑᨗ ᨈᨚᨇᨚ ᨈᨗᨀ᨞
ᨈᨚᨇᨚᨊ ᨕᨗ ᨄᨉ ᨕᨘᨒᨛ ᨑᨗ ᨔᨓ ᨆᨙᨁ᨞
ᨑᨗᨄᨚᨓᨓᨗᨊᨙᨊ ᨕᨗ ᨄᨉ ᨕᨘᨒᨛ ᨑᨗ ᨒ ᨕᨘᨑᨘ ᨇᨛᨔᨗ᨞
ᨔᨆᨔᨆ ᨕᨗ ᨆᨈᨙ ᨆᨂᨘᨑᨘ ᨕᨛᨔᨚ᨞
ᨒ ᨕᨘᨑᨘ ᨇᨛᨔᨗ ᨆᨈᨙ ᨑᨗ ᨔᨒᨗᨓᨛ᨞
ᨕᨗ ᨄᨉ ᨕᨘᨒᨛ ᨆᨈᨙ ᨑᨗ ᨒᨒᨛ᨞
ᨕᨗᨕ ᨊ ᨑᨚ ᨊᨑᨗᨕᨔᨛ ᨈᨘᨑᨘ ᨅᨙᨒ᨞ ᨕᨄ ᨔᨆᨔᨆ ᨕᨗ ᨆᨈᨙ᨞
ᨉᨘᨕ ᨕᨊᨊ᨞ ᨆᨀᨘᨋᨕᨗ ᨉᨘᨕ᨞
ᨕᨗᨕ ᨆᨌᨚᨕᨕᨙ ᨑᨗᨕᨔᨛ ᨕᨗ ᨕᨉᨗᨒᨘᨓᨘ᨞
ᨕᨗᨕ ᨆᨒᨚᨒᨚᨕᨙ ᨑᨗᨕᨔᨛ ᨓᨙ ᨕᨚᨄᨘ ᨔᨛᨂᨛ᨞
ᨑᨗᨑᨄᨊ ᨈᨚ ᨅᨙᨕᨘᨕᨙ ᨑᨗᨊᨕᨘᨑᨙᨊ᨞
ᨑᨗᨕᨔᨛᨂᨙ ᨓᨙ ᨈᨛᨋᨗᨍᨛᨒᨚ᨞
ᨉᨙ ᨕᨊᨊ᨞ ᨊᨑᨗᨕᨔᨛᨊ ᨆᨊᨂᨙ᨞
ᨒᨀᨕᨗᨊ ᨑᨗᨕᨔᨛ ᨒ ᨍᨓ ᨄᨔᨙ᨞
ᨆᨂᨘᨑᨘ ᨕᨅᨚ ᨕᨗ ᨓᨙ ᨄᨉ ᨕᨘᨒᨛ ᨓᨙ ᨈᨛᨋᨗᨍᨛᨒᨚ᨞ ᨉᨙ ᨊᨄᨉᨄᨉ᨞

ᨒᨚᨀᨊ ᨈᨚ ᨅᨙᨕᨘᨕᨙ ᨔᨗᨈᨗᨋᨚ ᨉᨘᨕᨉᨘᨕ ᨆᨄᨉ ᨆᨀᨘᨋᨕᨗ᨞
ᨔᨅ ᨆᨈᨙᨊ ᨕᨗᨉᨚᨊ᨞ ᨕᨅᨚᨊ ᨊᨑᨗᨕᨔᨛ ᨈᨚ ᨅᨙᨕᨘᨕᨙ᨞

ᨆᨙᨆᨊᨊ ᨆᨊᨛ ᨓᨓᨗᨊᨙᨊ ᨆᨊᨘᨑᨘᨂᨙ ᨑᨗ ᨒᨘᨓᨘ᨞

ᨕᨗᨕ ᨊᨕᨙ ᨄᨆᨘᨒᨊ ᨔᨘᨑᨛ ᨁᨒᨗᨁᨚᨕᨙ᨞
